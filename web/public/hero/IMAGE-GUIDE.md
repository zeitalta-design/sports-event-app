# ヒーロースライダー画像ガイド

## 配置場所

`public/hero/` に以下のファイル名で配置すると自動で反映される。

| ファイル名 | スライド |
|---|---|
| `entry-open.jpg` | 今エントリーできる大会 |
| `deadline-soon.jpg` | 締切間近の大会 |
| `beginner-friendly.jpg` | 初心者でも参加しやすい大会 |

画像がない場合はグラデーション背景でフォールバックする。

---

## 共通選定基準

### 必須条件

- 実写のみ使用する
- 文字入り画像は禁止（ゼッケン番号程度は許容）
- イラスト・バナー風の販促ビジュアルは禁止
- 人の動き・会場感・参加感があること
- UIの背景として使える落ち着いたトーンであること

### 避けるべきもの

- ただの風景だけの写真（人物がいない）
- 走者が横一列で走っているだけの単調な構図
- 派手すぎる・広告っぽいビジュアル
- 暗すぎて圧迫感のある写真
- 特定ブランドのロゴが大きく写っている写真

### 構図・技術要件

- 文字を置ける余白が左側にあること（テキストは左寄せ配置）
- モバイルで中央トリミング（object-cover）されても主題が残ること
- 横長の構図が適している（縦長は不可）
- 推奨解像度: 1920x1080 以上（Next.js が自動最適化する）
- ファイルサイズ: 500KB〜2MB 程度（大きすぎなければOK）

## スライド別選定基準

### 1. entry-open.jpg（今エントリーできる大会）

**狙い**: 活気・高揚感・これから始まる感

- スタート前後の雰囲気（ゲート前の集団、準備運動、号砲直後など）
- 複数人いて大会感がある
- 活気はあるが広告っぽすぎない
- 明るめのトーンが望ましい

**理想例**: スタートゲート前で準備している参加者たち、朝の受付風景
**NG例**: ゴール後の疲れた表情だけ、表彰式

### 2. deadline-soon.jpg（締切間近の大会）

**狙い**: 緊張感・本番感・決断を促す空気

- 会場導線やスタートゲートが見える
- 少し緊張感がある（スタート直前、コース上の走者など）
- 暗すぎず、煽りすぎない
- やや引きのアングルで会場全体が感じられると良い

**理想例**: スタートラインに並ぶ走者たち、コース上を力走する集団
**NG例**: 完全なクローズアップ、苦しそうな表情だけ

### 3. beginner-friendly.jpg（初心者でも参加しやすい大会）

**狙い**: 笑顔・仲間感・楽しさ・安心感

- 笑顔や仲間・家族で参加している雰囲気
- 初参加しやすそうに見える
- ガチすぎる競技写真は避ける
- 明るく温かみのあるトーン

**理想例**: ゴール後にハイタッチする参加者、家族連れで走っている風景、仮装ランナー
**NG例**: エリートランナーの激走、真剣な表情だけの写真

## 差し替え後の確認チェックリスト

画像を配置したら以下を確認する:

### 文字の可読性

- [ ] タイトル（白文字）が背景に埋もれず読めるか
- [ ] サブタイトル（白文字/半透明）も読めるか
- [ ] 画像が明るすぎて文字が見えにくい場合 → overlay の濃度を調整

### CTAボタン

- [ ] 白いCTAボタンが背景に埋もれていないか
- [ ] ボタンの位置に画像の明るい部分が重なっていないか

### トリミング・構図

- [ ] PC（横長）で主題が適切に表示されているか
- [ ] モバイル（中央トリミング）で人物の顔や主題が切れすぎていないか
- [ ] object-position の調整が必要なら SLIDES 配列の imagePosition を変更

### 3枚の統一感

- [ ] 3枚のトーン（明るさ・色味）が揃っているか
- [ ] 1枚だけ派手すぎたり暗すぎたりしないか
- [ ] スライド切替時に違和感がないか

### ブランド適合

- [ ] 広告バナーっぽく見えないか
- [ ] スポ活の信頼感・使いやすさのトーンを壊していないか
- [ ] 3秒で「スポーツ大会の検索サイト」と伝わるか

## 調整方法

画像配置後に微調整が必要な場合、以下のプロパティを SLIDES 配列で変更できる:

| プロパティ | 用途 | 例 |
|---|---|---|
| `imagePosition` | 画像のフォーカス位置 | `"center top"`, `"center 30%"` |
| `overlayOpacity` | オーバーレイの濃さ | `"from-black/70"` (暗め), `"from-black/50"` (明るめ) |

デフォルト値:
- imagePosition: `"center center"`（object-position）
- overlayOpacity: `"from-black/65 via-black/35 to-black/15"`（左から右へ薄くなるグラデーション）
